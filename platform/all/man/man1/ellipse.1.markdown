ellipse(1) -- truncate text intelligently
=========================================

## SYNOPSIS

`ellipse` <var>text</var> <var>length</var>

## DESCRIPTION

The ellipse(1) command truncates text to the nearest space within a specified length if it is longer, and appends a horizontal ellipse (â€¦) character.

## OPTIONS

* <var>text</var>:
  Text to render and, if too long, truncate.

* <var>length</var>:
  Maximum length to output.
